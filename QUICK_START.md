# 快速开始指南

本文档帮助您快速了解并使用校园新闻投稿系统。

## 系统简介

校园新闻投稿系统是一个专为校园媒体设计的内容管理平台，支持文字、摄影、视频等多种形式的投稿，并提供完整的审核发布流程。

## 用户角色

系统包含四种用户角色：

### 1. 超级管理员 (super_admin)
- 拥有系统所有权限
- 可以管理用户和权限
- 可以进行系统配置
- 可以查看所有日志

### 2. 管理员 (admin)
- 用户管理（不可删除用户）
- 稿件管理和发布
- 栏目管理
- 查看操作日志

### 3. 审核员 (editor)
- 审核分配的栏目稿件
- 初审或终审操作
- 查看审核历史

### 4. 投稿用户 (contributor)
- 注册并申请身份认证
- 通过认证后可投稿
- 查看自己的稿件状态
- 修改被退回的稿件

## 快速上手

### 对于投稿用户

#### 1. 注册账号
- 访问网站首页
- 点击"注册"
- 填写邮箱、用户名、密码
- 提交注册

#### 2. 身份认证
- 登录后进入个人中心
- 上传学生证或校园卡照片
- 填写学号、学院、专业等信息
- 提交认证申请
- 等待管理员审核（通常1-2个工作日）

#### 3. 开始投稿

**文字类稿件（新闻/文艺作品）**
- 点击"投稿"
- 选择"新闻"或"文艺作品"
- 选择具体栏目
- 填写标题和内容（可使用富文本编辑器）
- 或上传Word文档
- 可选：上传封面图
- 选择"立即投稿"或"保存草稿"

**多媒体稿件（摄影/视频）**
- 点击"投稿"
- 选择"摄影作品"或"视频作品"
- 填写作品标题
- 填写百度网盘分享链接
- 可选：填写提取码
- 上传封面图
- 填写作品简介
- 提交投稿

#### 4. 查看稿件状态
- 在"我的稿件"中查看所有投稿
- 状态说明：
  - 草稿：尚未提交
  - 待初审：已提交，等待初审
  - 审核中：初审通过，等待终审
  - 需修改：审核员要求修改
  - 已批准：终审通过，等待发布
  - 已发布：稿件已发布
  - 已拒绝：稿件被拒绝

#### 5. 修改稿件
- 如果稿件状态为"需修改"
- 点击"编辑"按钮
- 根据审核意见修改内容
- 重新提交

### 对于审核员

#### 1. 查看待审稿件
- 登录后访问"审核工作台"
- 查看分配给自己栏目的待审稿件
- 点击稿件查看详情

#### 2. 审核操作

**初审**
- 检查稿件基本质量
- 操作选项：
  - 通过：进入终审环节
  - 退回修改：要求作者修改（需填写修改意见）
  - 拒绝：不予录用（需填写拒绝理由）

**终审**
- 对初审通过的稿件进行终审
- 重点检查内容质量和安全性
- 操作选项同初审

#### 3. 填写审核意见
- 审核意见会通知作者
- 退回修改时，请明确指出需要修改的地方
- 拒绝时，请说明拒绝的理由

### 对于管理员

#### 1. 用户管理

**认证审核**
- 访问"用户管理" → "认证申请"
- 查看用户上传的认证文件
- 核对学号、学院等信息
- 操作：
  - 通过：用户获得投稿权限
  - 拒绝：说明拒绝原因（如照片模糊）

**角色分配**
- 在用户列表中找到用户
- 点击"编辑"
- 修改角色：
  - 审核员：需分配可审核的栏目
  - 管理员：获得管理权限（默认无删除权限）

**账号管理**
- 启用/禁用账号
- 查看用户投稿记录
- 查看用户操作日志

#### 2. 稿件管理

**发布稿件**
- 访问"稿件管理" → "待发布"
- 选择终审通过的稿件
- 点击"发布"
- 稿件状态变为"已发布"

**归档管理**（多媒体稿件）
- 将作者提供的网盘内容转存到官方网盘
- 点击稿件的"归档"按钮
- 填写官方归档链接
- 稿件标记为"已归档"

**紧急下线**
- 如发现已发布稿件存在问题
- 点击"下线"按钮
- 填写下线原因
- 稿件状态变为"已下线"

#### 3. 系统管理

**栏目管理**
- 添加/编辑/删除栏目
- 设置栏目类型和描述
- 分配栏目审核员

**系统设置**
- 网站名称
- 投稿限制：
  - 每日投稿上限
  - 每周视频投稿上限
- 文件上传限制
- SMTP邮件配置

**操作日志**
- 查看所有管理员和审核员的操作记录
- 按时间、用户、操作类型筛选
- 导出日志报表

## 投稿限制说明

### 默认限制
- 每日投稿上限：5篇
- 每周视频投稿上限：2个
- 单个文件最大：10MB

### 如何调整限制
管理员可在"系统设置"中调整投稿限制规则。

## 稿件审核流程

```
作者投稿
   ↓
待初审（审核员A）
   ↓
初审通过
   ↓
待终审（审核员B）
   ↓
终审通过
   ↓
待发布（管理员）
   ↓
已发布
   ↓
归档（可选，多媒体稿件）
```

任何环节都可以退回修改或拒绝。

## 通知机制

系统会在以下情况发送通知：

### 用户通知
- 认证审核结果
- 稿件初审结果
- 稿件终审结果
- 稿件发布通知

### 管理员通知
- 新的认证申请
- 待审稿件提醒
- 稿件积压警告

（注：邮件通知需配置SMTP）

## 常用功能

### 草稿保存
- 投稿时选择"保存草稿"
- 稿件不会进入审核流程
- 可随时编辑并提交

### 查询码
- 投稿成功后会生成查询码
- 可使用查询码匿名查询审核进度
- 适合不想公开身份的投稿

### 版本管理
- 每次修改稿件会生成新版本
- 可查看历史版本
- 可回退到旧版本（管理员）

## 安全提示

### 对于所有用户
1. 定期修改密码
2. 不要分享账号
3. 及时退出登录
4. 注意保护个人信息

### 对于管理员
1. 首次登录后立即修改默认密码
2. 妥善保管管理员权限
3. 定期查看操作日志
4. 定期备份数据

## 技巧与建议

### 投稿技巧
1. **标题**：简洁明了，突出重点
2. **内容**：结构清晰，逻辑连贯
3. **图片**：清晰美观，大小适中
4. **网盘链接**：确保链接长期有效

### 审核建议
1. **及时审核**：避免稿件积压
2. **意见明确**：退回时说明具体问题
3. **公平公正**：按照统一标准审核
4. **记录详细**：重要决定记录理由

### 管理建议
1. **定期备份**：每天备份数据库
2. **查看日志**：定期检查操作日志
3. **权限控制**：谨慎分配管理权限
4. **及时更新**：关注系统更新

## 常见场景

### 场景1：忘记密码
1. 联系管理员重置密码
2. 或使用邮箱找回功能（如已配置）

### 场景2：认证被拒绝
1. 查看拒绝原因
2. 准备清晰的认证文件
3. 重新提交认证申请

### 场景3：稿件被退回
1. 查看审核意见
2. 根据要求修改内容
3. 重新提交审核

### 场景4：网盘链接失效
1. 及时更新网盘链接
2. 联系管理员更新归档链接
3. 建议使用永久链接
